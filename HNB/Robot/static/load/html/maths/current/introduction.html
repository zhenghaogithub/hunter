STS，SQS 定义：

$\frac{b}{a}$    Let X be a set of v elements. A t-wise balanced design (t-BD) of order v is a pair (X, Æ) such that each t-subset of X is contained in exactly one block of Æ.. An S(t, K, v) denotes a t-BD of order v with block sizes from the set K. An S(2,{3}, v) is also called Steiner triple system of order v (STS(v)), while an S(3,{4}, v) is called Steiner quadruple system of order v (SQS(v)).

   令 X 是一个 v 元集合，一个v阶的 t 平衡设计(t-BD)是一个 对子 (X, Æ)，如果 X 中的任意一个t元子集都包含在Æ的一个区组中. 如果一个v阶的 t 平衡设计(t-BD)区组的大小组成一个集合K,我们记为S(t, K, v).如果v阶的 t 平衡设计(t-BD)所有的区组大小都相等，也就是集合K中只有元素 k，我们记为S(t, {k}, v).当t = 2，k = 3，我们称S(2, {3}, v)为斯坦纳三元系，简记为STS(v) .我们称S(3, {4}, v)为斯坦纳四元系 ,简记SQS(v)

LSTS 定义：

   Two STS(v)s on the same set X are said to be disjoint if they have no triples in common. A set of v − 2 pairwise disjoint STS(v)s on the same set X is called a large set of STS(v)s and denoted by LSTS(v).

   两个定义在集合X上的STS称为不相交的，如果他们没有相同的三元组.v-2个两两不相交的STS的集合称为一个LSTS大集，记为LSTS(v)

GSTS,GLS定义：

   Suppose that (X,Æ) is an STS(v) and a, b are two distinct elements of X. Let {a, b, c} be the unique triple in B containing a and b. If there exists a diagraph D on X \ {a, b, c} with every vertex of indegree and outdegree 1 and if its underlying undirected graph has edge set {{x, y} : x, y ∈ X \ {a, b, c},{a, x, y} ∈Æ or {b, x, y} ∈ Æ}, then the 5-tuple (X, a, b, Æ , D) is called a good STS(v) and denoted by GSTS(v). A good large set of STS(v)s, denoted by GLS(v), is a collection {(X, a, b, Æ_i, D_i) : 1 ≤ i ≤ v − 2} provided that each (X, a, b, Æ_i, D_i) is a GSTS(v), {(X, Æ_i) : 1 ≤ i ≤ v − 2} is an LSTS(v) and ∪1≤i≤v−2 D_i = {(x, y) : x, y ∈ X \ {a, b}, x!= y}.

   假设(X,Æ)是一个STS(v),a,b是X中两个不同的元素,Æ中含有唯一的区组 { a,b,c }.如果存在一个定义在X\{a,b,c}上的有向图D,它每个点的 出度,入度都是1,且它的无向图的边集满足 {{x, y} : x, y ∈ X \ {a, b, c},{a, x, y} ∈Æ or {b, x, y} ∈ Æ},那么这个五元组(X, a, b, Æ , D) 称为一个好的 STS,简记为 GSTS(v) .如果存在集合{(X, a, b, Æ_i , D_i),1 ≤ i ≤ v − 2},并且满足{(X, Æ_i) : 1 ≤ i ≤ v − 2} 是一个 LSTS(v) 和 ∪1≤i≤v−2 D_i = {(x, y) : x, y ∈ X \ {a, b}, x!= y}.那么称{(X, a, b, Æ_i , D_i),1 ≤ i ≤ v − 2}是一个好的大集,记为 GLS(v) .
